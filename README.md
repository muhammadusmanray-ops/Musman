Global Health Data Scraper (Ultra Pro)
Fast, Reliable, and Automated Medical Data Extraction
This Actor is a professional-grade tool designed to extract structured data from global health websites, medical journals, and WHO portals. It provides researchers and data scientists with high-quality datasets in JSON, CSV, or Excel formats.
Key Features:
Automatic Sync: Fully integrated with GitHub for instant updates and deployments.
Advanced Input Schema: Users can easily define target URLs and limit results.
Proxy Support: Built-in proxy configuration to avoid getting blocked by medical sources.
High Reliability: 100% success rate on the latest builds and runs.
How to Use:
Enter the target medical or health website URLs in the input section.
Click the Start button to begin the scraping process.
Download your structured data from the Dataset tab once the run is succeeded.
